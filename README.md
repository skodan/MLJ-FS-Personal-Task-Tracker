# Personal Task Tracker

A **full-stack task manager** built from scratch to learn modern web development.

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://react.dev)
[![Flask](https://img.shields.io/badge/Flask-000000?style=for-the-badge&logo=flask&logoColor=white)](https://flask.palletsprojects.com)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-316192?style=for-the-badge&logo=postgresql&logoColor=white)](https://www.postgresql.org)

---

## Features
- Add, edit, delete, and complete tasks
- Data persists with **PostgreSQL**
- Responsive UI with **Tailwind CSS**
- REST API ready for **React Native mobile app**
- Deployed on **Netlify** (frontend) + **Render.com** (backend)

---

## Tech Stack

| Layer     | Technology                  |
|---------|-----------------------------|
| Frontend | React, Tailwind CSS         |
| Backend  | Python, Flask, SQLAlchemy   |
| Database | PostgreSQL                  |
| Deploy   | Netlify, Render.com         |
